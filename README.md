## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NPM updated
- Modern web browser
- Git

### Setup

    git clone https://github.com/Fourteen98/toolhub-frontend.git

### Install
    npm install

### Usage
    npm run dev


### Deployment
    npm run deploy
